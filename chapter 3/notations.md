## Notations

| **Syntax**	       | **Description**                  |
|--------------------|----------------------------------|
| 𝜌                  | env                              |
| [ ]                | Empty env                        |
| [var = val] 𝜌      | (extend-env var val 𝜌)           |
| «exp»              | AST for expression exp           |
| ⌈n⌉                | (num-val n)                      |
| ⌊n⌋                | (expval->num val)                |

### Note: $⌈⌊n⌋⌉=n$
